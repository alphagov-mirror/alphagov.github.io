Government Digital Service

The UK Government Digital Service has a practice of Coding In The Open. That means a lot of public repositories end up in our GitHub account. Those repositories can be hard to navigate so we're assembling a guide to it here, identifying the pieces we think are likely to be useful to others and then the projects that other code is focussed on.

Libraries & Utilities

  • Unicorn Herder

    python Unicorn Herder is a utility designed to assist in the use of Upstart and similar supervisors with Unicorn.

  • Fake Google Analytics

    ruby Fake Google Analytics (GA) is a tool used to shorten the feedback loop when testing the interaction of a website with Google Analytics.

  • Character Encoding Cleaner

    ruby This script is for fixing illegal encodings in text files. It's designed to clean up corrupt character sequences in UTF-8 files.

  • Google Auth Bridge

    Google allow you to authenticate with their APIs with OAuth.

  • Magna Charta

    javascript A javascript library (currently a jQuery plugin) for producing bar charts from tables.

  • PassphraseEntropy

    ruby Estimate the entropy of a passphrase. Inspired by xkcd 936.

  • Rack::Geo

    ruby Simple Rack middleware which processes incoming requests and adds, updates, and passes through as a Header, Geo-location information

  • SBV to TTAF

    ruby Convert SBV caption files to the Timed Text Authoring Format XML schema used by the Nomensa accessible video player.

Transitioning to GOV.UK

We built a set of tools to support the transition from DirectGov, BusinessLink and a host of other websites onto GOV.UK. You can read an overview of that in No Link Left Behind and Testing The Redirections. The key chunks of code were:

  • Migratorator

    Allows multiple users to create one-to-one mappings for each URL, where the mapping consists of the source URL, status (whether it will be redirected or whether, no longer representing a user need, it is now gone) and, if applicable, the page to which it will be redirected.

  • Review-o-matic

    We needed to confirm that the mappings were actually correct. We wanted several people to check each mapping, so we created the Review-O-Matic.

  • Side-by-side Browser (aka Review-o-matic-Explorer)

    The Side-by-side browser is a Node.js proxy that serves itself and the site being reviewed on the same domain, so that it's 'live' and not blocked by the Same-Origin policy.

  • The Redirector

    The Redirector project drives the process of generating the Nginx configuration for redirection.

Performance Platform

The Performance Platform team creates dashboards to show the performance of government over time. We occasionally post on the Data at GDS blog.

  • Spotlight

    Spotlight is the Node.js frontend which replaces and unifies the other frontend apps (Limelight, Datainsight Frontend and Transactions Explorer) for the Performance Platform.

  • Backdrop

    Backdrop is the Python API used for storing and exposing data stored in the Performance Platform.

  • Puppet

    We use Puppet to provision machines in all our environments.

Tools for the office

We have a lot of code that shows up on displays around our office.

  • Fourth Wall

    Fourth Wall is a monitor that shows open pull requests on GitHub (both public and Enterprise) and build status on Travis.

  • Showtime

    Showtime is a Ruby project that can automate a web browser to show a journey through a website.

A little background

Who is this code by?

This is the GitHub account of the Government Digital Service, a unit of the UK government's Cabinet Office tasked with delivering digital by default. For more on GDS and the UK government's digital strategy see the GDS blog.

Why alphagov?

Alpha.gov.uk was the proof of concept that demonstrated that the GOV.UK project (and the Government Digital Service with it) could work. It was launched in May 2011 and remained live until the beta of GOV.UK launched at the end of January 2012. This GitHub account was set up for that project and then developed to be the main account for the Government Digital Service.
